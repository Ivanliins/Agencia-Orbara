Crie um site institucional completo, moderno e VISUALMENTE IMPACTANTE para a agência de publicidade "Orbara".

===========================
SOBRE A ORBARA
===========================
A Orbara é uma agência boutique de publicidade focada em sites institucionais de alta conversão. Unimos Google Ads + SEO + copywriting estratégico para transformar o site do cliente em uma máquina silenciosa de vendas, aprimorando a comunicação entre o prestador de serviço e o cliente final.

Tagline: "Onde marcas encontram sua gravidade."
Posicionamento: boutique, ousada, confiante, cósmica.
Inspiração estética: design bold contemporâneo com fontes gigantes, shapes arredondados (pill shapes), contraste brutal, estilo editorial moderno.

===========================
STACK TÉCNICA
===========================
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (animações sutis)
- Lucide React (ícones)
- Formulário com react-hook-form
- SEO completo: metadata dinâmica, Open Graph, sitemap.xml, robots.txt, schema.org JSON-LD (Organization)
- Fontes otimizadas via next/font
- Responsivo mobile-first
- Performance Lighthouse 90+

===========================
IDENTIDADE VISUAL (MUITO IMPORTANTE - SEGUIR À RISCA)
===========================

PALETA DE CORES — criar como CSS variables no globals.css:
- Orbit Orange: #ff5d00 (cor primária, CTAs, destaques, grandes blocos de cor)
- Deep Black: #0d0101 (fundo principal escuro, textos fortes)
- Starlight: #fffafa (off-white, fundos claros, textos sobre preto/laranja)

TIPOGRAFIA — USAR APENAS A FONTE ONEST (carregar via Google Fonts ou next/font/google):
- Carregar Onest nos pesos 300, 400, 500, 600, 700, 800, 900
- Headlines/títulos: Onest peso 800 ou 900, TAMANHOS GIGANTES (clamp(4rem, 12vw, 12rem) nos heros)
- Subheadlines: Onest peso 500, tamanhos médios
- Corpo: Onest peso 400
- Line-height apertado nos títulos (0.9 a 1.0) para máximo impacto
- Tracking levemente negativo nos títulos grandes (-0.02em)

ELEMENTOS VISUAIS CHAVE:
- SHAPES ARREDONDADOS ESTILO PÍLULA: muitos elementos com border-radius grande (rounded-full ou rounded-[40px]/[60px]) criando formas estilo "pill". Blocos de cor laranja com cantos arredondados enormes, tipo cápsulas gigantes.
- Cards e seções com border-radius de 32px, 40px ou até 60px
- Contraste brutal: blocos inteiros laranja (#ff5d00) alternando com blocos pretos (#0d0101) e off-white (#fffafa)
- Fontes gigantes quebrando o layout, invadindo bordas das seções
- Animações sutis: fade-in no scroll, leve parallax, hover scale sutil nos botões
- Micro-interações: botões com transição de cor suave, cursor pointer customizado opcional

LOGO ORBARA:
- Símbolo: uma órbita (círculo fino) com um ponto luminoso na circunferência superior direita
- Wordmark "ORBARA" em Onest peso 800, tracking amplo
- Criar como componente SVG inline, adaptável às cores de fundo

===========================
REGRAS DE DESIGN INEGOCIÁVEIS
===========================
1. Fontes SEMPRE gigantes nos títulos principais (mínimo clamp(3rem, 10vw, 10rem))
2. Border-radius grandes em quase todos os blocos (nunca menos de 24px em cards, geralmente 40-60px)
3. Contraste extremo entre seções: preto → laranja → off-white → preto (alternância ousada)
4. Respiração generosa: muito padding vertical, nunca apertar conteúdo
5. Onest é a ÚNICA fonte do site
6. Botões sempre em formato pill (rounded-full)
7. Texto preto sobre laranja e laranja sobre preto — nunca outras combinações exóticas
